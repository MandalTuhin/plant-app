A simple full-stack plant tracking app built with:

- React (frontend)
- Express.js + Node (backend)
- MongoDB (via Docker)
- Image uploads (via multer)
- JWT-based authentication (optional)

## How to Run

1. Start MongoDB container:

   ```bash
   sudo docker start plant-mongo
   ```

2. Start backend

   ```bash
   cd backend
   nodemon index.js
   ```

3. Start frontend

   ```bash
   cd frontend
   npm start
   ```
